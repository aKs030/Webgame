# To-Do Liste

Eine moderne Aufgabenverwaltung, die hilft, den Überblick zu behalten.

## Features
- Aufgaben hinzufügen, abhaken und löschen
- Filter für "Alle", "Offen" und "Erledigt"
- Automatische Speicherung (LocalStorage)
- Statistik-Anzeige
- Bulk-Actions (Erledigte löschen, Alles löschen)

## Technologien
- HTML5
- CSS3 (Flexbox, Glassmorphism)
- Vanilla JavaScript
