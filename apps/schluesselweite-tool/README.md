# Schlüsselweite Tool

Ein einfaches Tool zur Berechnung der Schlüsselweite (SW) für metrische Schrauben und Muttern.

## Features
- Unterstützung für ISO und DIN Normen.
- Schnellauswahl gängiger Größen (M3 - M24).
- Detaillierte Informationen zu Steigung und Durchgangsloch.
- Modernes, responsives Design.
- Optimiert für die Integration in Portfolio-Cards.

## Verwendung
Wähle einfach die gewünschte Größe aus der Liste oder gib sie manuell ein (z.B. "M12"). Das Tool zeigt sofort die passende Schlüsselweite an.
