# Zahlen Raten

Ein klassisches Logik-Spiel, bei dem man eine zufällige Zahl erraten muss.

## Features
- Generiert Zufallszahl zwischen 1 und 100
- Visuelles Feedback (Zu hoch / Zu niedrig)
- Versuchs-Zähler und Historie
- Bereichsanzeige der bisherigen Versuche
- Modernes Design

## Technologien
- HTML5
- CSS3
- Vanilla JavaScript
