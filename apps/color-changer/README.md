# Color Changer

Ein nützliches Tool für Designer und Entwickler, um schnell Farben zu generieren und zu verwalten.

## Features
- Generiert zufällige Hex-Farbcodes
- Live-Vorschau in großer Fläche
- 1-Klick Kopieren in die Zwischenablage
- Verlauf der letzten 15 Farben (LocalStorage)
- "Weiß"-Preset für schnellen Reset

## Technologien
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
