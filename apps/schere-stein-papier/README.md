# Schere Stein Papier

Ein klassisches Schere-Stein-Papier Spiel, implementiert als leichte Single-Page-Application.

## Features
- Interaktives Gameplay gegen den Computer
- Echtzeit-Punktestand (Spieler vs. Computer vs. Unentschieden)
- Animierte Auswahl
- Responsive Design (Mobile First)
- Modernes "Glassmorphism" UI

## Technologien
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (Keine Frameworks)
